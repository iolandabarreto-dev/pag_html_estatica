## 📌 Sobre o projeto

O **Coding — Desenvolvimento Web** é um site institucional que simula uma empresa de tecnologia especializada em criação de soluções digitais.

O projeto foi desenvolvido com foco em **prática de front-end**, organização de layout e construção de uma interface moderna e responsiva, apresentando serviços, equipe e canais de contato.

📚 **Este projeto foi proposto pela AdaTech**, como parte do programa **CaixaVerso — Desenvolvimento Front-End Nível 1**.

---

## 🎯 Objetivo

Este projeto tem como objetivo:

* Praticar estruturação com HTML semântico
* Aplicar conceitos de responsividade com CSS
* Criar uma interface profissional de apresentação
* Simular um ambiente real de empresa de desenvolvimento

---

## ✨ Funcionalidades

✔️ Navegação por seções (Home, Serviços, Sócios e Contato)
✔️ Apresentação visual dos serviços oferecidos
✔️ Seção de equipe com links externos
✔️ Integração com Google Maps
✔️ Formulário de contato
✔️ Menu responsivo (mobile)
